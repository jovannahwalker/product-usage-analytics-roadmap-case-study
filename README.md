# Product Usage Analytics Roadmap Case Study

This repository contains a redacted product analytics case study demonstrating how raw product usage data (DAU/MAU, retention, and feature usage) was translated into a concrete roadmap recommendation for product leadership.[web:23][web:24]

## Overview

- **Focus:** Early-life engagement, activation, DAU/MAU stickiness, and cohort retention
- **Goal:** Show how product usage data can drive a specific roadmap theme (onboarding and activation)
- **Audience:** Hiring managers, product leaders, and teams evaluating product/data analytics skills

## Repository Structure

- `case-study.md` – Full redacted case study (business context, analysis, findings, and roadmap)
- `README.md` – High-level overview you are reading now
- (Optional) `notebooks/` – Example SQL/Python or analysis notebooks (if you decide to add them)
- (Optional) `charts/` – Simple charts for DAU/MAU and cohort retention (PNG or SVG)

## Skills Demonstrated

- Defining meaningful active-user metrics (beyond raw logins) and DAU/MAU for stickiness
- Building and interpreting cohort retention curves to find early drop-off points
- Connecting activation behavior to long-term retention
- Translating analysis into a prioritized roadmap recommendation with clear success metrics
- Communicating findings in a concise, leadership-ready narrative

## Tools and Methods

- SQL for event extraction and cohort creation
- Excel or Python for retention and segmentation analysis
- Cohort analysis, funnel analysis, activation analysis, and trend monitoring
- Dashboarding for ongoing KPI tracking
